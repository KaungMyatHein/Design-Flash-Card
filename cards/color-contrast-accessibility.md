# Color Contrast & Accessibility
# Color Contrast နဲ့ Accessibility (အရောင်ခြားနားမှုနဲ့ လက်လှမ်းမီမှု)

## Question
Why is color contrast important for accessibility?

## Question_MM
Accessibility အတွက် color contrast က ဘာကြောင့် အရေးကြီးတာလဲ?

## Answer
Good color contrast ensures text and UI elements are readable for all users — including those with color blindness, low vision, or using devices in bright sunlight. It's the most basic and impactful accessibility measure.

## Answer_MM
Color contrast ကောင်းရင် user အားလုံး — colorblind, မျက်စိအားနည်း သို့မဟုတ် နေပူထဲ ဖုန်းကြည့်နေသူတွေ အပါအဝင် — ဖတ်နိုင်တယ်။ Accessibility အတွက် အခြေခံဆုံးနဲ့ အထိရောက်ဆုံး။

## Why
- WCAG has 2 standards: AA (minimum) and AAA (better)
- Body text needs 4.5:1 ratio (AA); large text 18pt+ needs 3:1
- 1 in 12 men are colorblind — can't rely on color alone for info
- Inclusive design benefits ALL users, not just disabled users
- Test on real devices — looks fine on laptop doesn't mean fine on phone

## Why_MM
- WCAG standard ၂ ခုရှိတယ် — AA (အနည်းဆုံး) နဲ့ AAA (ပိုကောင်း)
- Body text: 4.5:1 ratio လို၊ Large text (18pt+): 3:1 ratio လိုတယ်
- အမျိုးသား ၁၂ ယောက်မှာ ၁ ယောက် colorblind — အရောင်တစ်ခုတည်းနဲ့ info ပေးလို့ မရဘူး
- Inclusive design က disabled user ပဲ မဟုတ်ဘူး user အားလုံးကို ကောင်းစေတယ်
- Laptop ပေါ်မှာ ကောင်းပေမယ့် ဖုန်းပေါ် မကောင်းနိုင်ဘူး — တကယ့် device ပေါ်မှာ test လုပ်

## Example
- Colour Contrast Analyser tool for checking AA/AAA compliance
- Using icons + color together (not color alone) for status indicators
- Dark text on light background for body copy
- Testing designs on actual devices in non-ideal lighting

## Example_MM
- Colour Contrast Analyser tool နဲ့ AA/AAA ပြည့်မပြည့် စစ်တယ်
- Status indicator မှာ အရောင်ပဲ မဟုတ်ဘဲ icon လေးပါ ထည့်ပေးတယ်
- Body copy မှာ light background ပေါ် dark text သုံးတယ်
- တကယ့် device ပေါ်မှာ နေရောင်ထဲ ထိုင်ပြီး test လုပ်တယ်
