# Status Quo Bias
# Status Quo Bias (ရှိရင်းစွဲကို ဆက်လိုချင်မှု)

## Question
Why do people tend to stick with the default option even when better alternatives exist?

## Question_MM
ဘာကြောင့် လူတွေက ပိုကောင်းတဲ့ alternative ရှိပေမယ့် default option မှာပဲ ဆက်နေချင်ကြတာလဲ?

## Answer
Status quo bias is our tendency to prefer things as they are. Like Newton's law of inertia applied to humans — we stay at rest unless pushed to change. This is why default settings are so powerful and why subscription services auto-renew.

## Answer_MM
Status quo bias ဆိုတာ ရှိနေတဲ့ အတိုင်း ဆက်ထားချင်တဲ့ သဘောထား ဖြစ်တယ်။ Newton ရဲ့ inertia law ကို လူတွေမှာ apply လုပ်သလိုပဲ — ပြောင်းခိုင်းမှ ပြောင်းတယ်။ ဒါကြောင့် default setting တွေ ဒီလောက် powerful ဖြစ်တာ၊ subscription service တွေ auto-renew ဖြစ်တာ။

## Why
- Defaults are incredibly powerful — most users never change them
- Change requires effort and introduces risk of making things worse
- Combined with loss aversion, changing feels like losing something familiar
- Opt-out is more effective than opt-in for user participation
- Smart defaults can nudge users toward better outcomes

## Why_MM
- Default တွေ ဘယ်လောက် powerful လဲဆိုတော့ user အများစု ဘယ်တော့မှ မပြောင်းဘူး
- ပြောင်းဖို့ effort လိုတယ်၊ ပိုဆိုးသွားမှာ ဆိုတဲ့ risk ပါတယ်
- Loss aversion နဲ့ ပေါင်းရင် ပြောင်းတာက familiar အရာ ဆုံးရှုံးသလို ခံစားရတယ်
- Opt-out က opt-in ထက် user participation ပိုများတယ်
- Smart default တွေက user တွေကို ပိုကောင်းတဲ့ outcome ဆီ nudge လုပ်နိုင်တယ်

## Example
- iPhone notification defaults are ON — most users never change them
- Netflix auto-plays the next episode — almost nobody turns this off
- Chrome ships with Google as default search — few people switch to Bing
- Cookie consent banners highlight "Accept All" as the default action

## Example_MM
- Myanmar user အများစု phone ရဲ့ default ringtone ကို ဘယ်တော့မှ မပြောင်းဘူး — status quo bias
- KBZ Pay default language English ထားရင် Myanmar ပြောင်းတဲ့သူ နည်းတယ် — default ရှိတဲ့အတိုင်း သုံးတယ်
- Viber ကို Myanmar user တွေ အရင်သုံးခဲ့လို့ Telegram ပိုကောင်းပေမယ့် ပြောင်းဖို့ ခက်တယ်
- Myanmar online shop တွေ COD (Cash on Delivery) default ထားရင် user တွေ ငွေသာပေးတယ် — online payment option ရှိပေမယ့် default မှာ ဆက်နေတယ်
