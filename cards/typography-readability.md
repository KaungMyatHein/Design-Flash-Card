# Typography & Readability
# Typography နဲ့ ဖတ်ရလွယ်ကူမှု

## Question
What are the key typography rules that make text easy to read on screens?

## Question_MM
Screen ပေါ်မှာ text ဖတ်ရလွယ်အောင် လိုက်နာသင့်တဲ့ typography rule အဓိက ဘာတွေလဲ?

## Answer
Good screen typography needs three things in balance: proper text size (16px minimum for body), comfortable line height (about 140% of text size), and the right line length (around 10 words per line). These three work together to create readable, comfortable text.

## Answer_MM
Screen typography ကောင်းဖို့ သုံးခု balance ညီဖို့ လိုတယ် — text size မှန်ကန်ရမယ် (body text အနည်းဆုံး 16px)၊ line height သက်တောင့်သက်သာ ရှိရမယ် (text size ရဲ့ 140% လောက်)၊ line length မှန်ရမယ် (တစ်ကြောင်း စာလုံး 10 လုံးလောက်)။ ဒီသုံးခုက အတူတူ အလုပ်လုပ်ပြီး ဖတ်ရလွယ်တဲ့ text ဖန်တီးတယ်။

## Why
- Line height too tight makes lines blur together; too loose breaks reading flow
- Longer lines need more line height to help eyes find the next line start
- Body text below 16px strains eyes on most devices
- Vertical rhythm between paragraphs maintains reading momentum
- Responsive typography must scale proportionally across screen sizes

## Why_MM
- Line height ကျပ်လွန်းရင် line တွေ ရောနေတယ်၊ ဟလွန်းရင် reading flow ပြတ်တယ်
- Line ရှည်ရင် line height ပိုလိုတယ် — မျက်လုံးက နောက်ကြောင်း အစ ရှာရလွယ်အောင်
- Body text 16px အောက်ဆို device အများစုမှာ မျက်လုံးပင်ပန်းတယ်
- Paragraph ကြား vertical rhythm က reading momentum ထိန်းပေးတယ်
- Responsive typography က screen size အလိုက် proportional ပြောင်းရတယ်

## Example
- Medium uses perfect line height so articles feel comfortable even during long reads
- Apple's website uses 17px body text with generous 1.5 line height
- The New York Times limits article line length to ~65 characters for optimal readability
- Notion uses 1.2rem heading line height, tighter than its body text

## Example_MM
- Myanmar font (Pyidaungsu, Padauk) က English font ထက် line height ပိုလိုတယ် — Myanmar စာလုံးတွေ vertically ပိုကြီးလို့
- Irrawaddy News website မှာ Myanmar article text size ကို ကြီးကြီးထားတယ် — Myanmar script ဖတ်ရလွယ်အောင်
- Facebook Myanmar post တွေမှာ Zawgyi/Unicode font issue ကြောင့် readability ထိခိုက်တယ် — font consistency အရေးကြီးတယ်
- Myanmar e-commerce site တွေမှာ product description ကို column width ကျဉ်းကျဉ်းထားရင် Myanmar text ဖတ်ရခက်တယ်
