# The Hook Model
# Hook Model (အလေ့အထဖြစ်စေသော ပုံစံ)

## Question
What is the Hook Model and what are its four phases?

## Question_MM
Hook Model ဆိုတာ ဘာလဲ၊ အဆင့် ၄ ဆင့် ဘာတွေလဲ?

## Answer
The Hook Model is a framework by Nir Eyal for building habit-forming products. Its four phases are: Trigger (what prompts the user), Action (simplest behavior), Variable Reward (unpredictable payoff), and Investment (user puts something in).

## Answer_MM
Nir Eyal ရဲ့ habit ဖြစ်စေတဲ့ product တည်ဆောက်ဖို့ framework။ အဆင့် ၄ ခု: Trigger (လှုံ့ဆော်တာ)၊ Action (အလွယ်ဆုံး behavior)၊ Variable Reward (မထင်မှတ်ထားတဲ့ ဆုလာဘ်)၊ Investment (user က တစ်ခုခု ထည့်တာ)။

## Why
- Trigger: external (notification, email) → internal (bored, curious) over time
- Action: motivation + ability combined → must be the easiest possible behavior
- Variable Reward: unpredictable payoff creates craving (slot machine effect)
- Investment: the more users put in (data, content, followers), the harder it is to leave
- Repeat this cycle enough times and it becomes a habit

## Why_MM
- Trigger: အရင်က notification လို external trigger ကနေ နောက်ဆုံး ပျင်းတာ/စိတ်ဝင်စားတာလို internal trigger ဖြစ်လာတယ်
- Action: motivation နဲ့ ability ပေါင်းပြီး တတ်နိုင်သမျှ လွယ်ဆုံး action ဖြစ်ရမယ်
- Variable Reward: ဘာရမယ် ကြိုမသိရတာက "ထပ်ကြည့်ချင်တယ်" ဆိုတဲ့ craving ဖြစ်စေတယ် — slot machine effect
- Investment: data, content, followers ထည့်ထားလေ ထွက်ခဲ့လေ
- ဒီ cycle ထပ်ခါထပ်ခါ ပတ်ရင် habit ဖြစ်လာတယ်

## Example
- Instagram: bored (trigger) → open app (action) → new likes/posts (variable reward) → post photo (investment)
- Pinterest: see interesting pin (trigger) → click (action) → discover related pins (reward) → save pins (investment)
- Email: notification (trigger) → check inbox (action) → new messages (reward) → reply (investment)

## Example_MM
- Instagram: ပျင်းတယ် → app ဖွင့် → like/post အသစ်တွေ ရှိတယ် → ဓာတ်ပုံတင်
- Pinterest: pin စိတ်ဝင်စားတယ် → နှိပ်ကြည့် → ဆက်စပ် pin တွေ တွေ့ → pin save လုပ်
- Email: notification လာတယ် → inbox ကြည့် → message အသစ် ရှိတယ် → reply ပြန်
