# IKEA Effect
# IKEA Effect (ကိုယ်တိုင်လုပ်ရင် ပိုတန်ဖိုးထားတဲ့ သက်ရောက်မှု)

## Question
Why do users value things more when they've put effort into creating or customizing them?

## Question_MM
ဘာကြောင့် user တွေက ကိုယ်တိုင် ဖန်တီးတာ ဒါမှမဟုတ် customize လုပ်ထားတဲ့ အရာကို ပိုတန်ဖိုးထားကြတာလဲ?

## Answer
The IKEA Effect shows that when people invest labor into something — assembling furniture, customizing a profile, building a playlist — they develop an irrational love for it. The effort itself creates emotional attachment and higher perceived value.

## Answer_MM
IKEA Effect အရ လူတွေ တစ်ခုခုမှာ labor invest လုပ်ရင် — furniture assemble လုပ်တာ၊ profile customize လုပ်တာ၊ playlist ဆောက်တာ — irrational love ဖြစ်လာတယ်။ Effort ကိုယ်တိုင်က emotional attachment နဲ့ perceived value ကို မြင့်စေတယ်။

## Why
- Users who invest effort feel ownership, which triggers the endowment effect
- Customization creates a sense of "this is uniquely mine"
- Small efforts compound — the more invested, the harder to leave
- Products that leverage user effort create stickier experiences
- Past behavior predicts future behavior — effort creates commitment

## Why_MM
- Effort invest လုပ်ထားတဲ့ user က ownership ခံစားရပြီး endowment effect trigger ဖြစ်တယ်
- Customization က "ဒါ ငါ့ဟာ တစ်ခုတည်း" ဆိုတဲ့ ခံစားချက် ဖန်တီးတယ်
- Effort အနည်းငယ်တွေ စုပြီး ပိုများလာလေ ထွက်ခွာဖို့ ပိုခက်လာလေ
- User effort ကို leverage လုပ်တဲ့ product တွေ stickier ဖြစ်တယ်
- အရင် behavior က future behavior ကို predict လုပ်တယ် — effort က commitment ဖန်တီးတယ်

## Example
- Pinterest users can't leave because of years of carefully curated boards
- Notion users resist switching tools after setting up custom workspace templates
- Spotify playlist builders feel deep attachment to their hand-picked collections
- Roblox and Minecraft let players build worlds, making the game impossible to quit

## Example_MM
- Myanmar Facebook user တွေ album တွေ post တွေ နှစ်ပေါင်းများစွာ ဆောက်ထားလို့ account ဖျက်ဖို့ စိတ်မပါဘူး
- Canva မှာ template တွေ ကိုယ်တိုင် customize ထားတဲ့ Myanmar designer တွေ Figma ပြောင်းဖို့ ခက်တယ်
- Viber group chat တွေ ကိုယ်တိုင် create လုပ်ပြီး member စုထားလို့ Telegram ပြောင်းဖို့ "ငါစုထားတာတွေ" ဆုံးမှာ ကြောက်တယ်
- Myanmar gamer တွေ Mobile Legend rank ကို ကိုယ်တိုင် grind ထားလို့ game ဖျက်ရင် effort အကုန်လုံး ဆုံးမယ်ဆိုပြီး ဆက်ကစားတယ်
