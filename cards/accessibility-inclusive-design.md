# Accessibility & Inclusive Design
# Accessibility နဲ့ လူတိုင်းအတွက် ဒီဇိုင်း

## Question
Why does designing for accessibility end up making the product better for everyone?

## Question_MM
Accessibility အတွက် design လုပ်တာက ဘာကြောင့် လူတိုင်းအတွက် product ပိုကောင်းလာစေတာလဲ?

## Answer
Designing for people with special needs — larger text, higher contrast, keyboard navigation — often benefits ALL users. In dim lighting, everyone needs high contrast. With busy hands, everyone needs voice control. Accessibility features become universal improvements.

## Answer_MM
Special needs ရှိတဲ့ သူတွေအတွက် design လုပ်တာ — text ကြီးကြီး၊ contrast မြင့်မြင့်၊ keyboard navigation — က user အားလုံးကို အကျိုးပြုတယ်။ အလင်းမှိန်ရင် လူတိုင်း contrast မြင့်တာ လိုတယ်။ လက်အားလပ်မရှိရင် လူတိုင်း voice control လိုတယ်။ Accessibility feature တွေက universal improvement ဖြစ်လာတယ်။

## Why
- Screen reader users scan with their ears, just like sighted users scan with eyes
- Keyboard-accessible design benefits power users who prefer shortcuts
- High contrast helps everyone in bright sunlight or dim rooms
- Clear content structure benefits both screen readers and SEO
- Adding alt text to images helps when images fail to load too

## Why_MM
- Screen reader user တွေက နားနဲ့ scan လုပ်တယ်၊ sighted user တွေ မျက်လုံးနဲ့ scan လုပ်သလိုပဲ
- Keyboard-accessible design က shortcut ကြိုက်တဲ့ power user တွေကိုလည်း အကျိုးပြုတယ်
- High contrast က နေရောင်ကြီးတဲ့အခါ၊ အခန်းမှိန်တဲ့အခါ လူတိုင်းကို ကူတယ်
- Content structure ကောင်းရင် screen reader ရော SEO ရော အကျိုးရှိတယ်
- Image alt text က image load မဖြစ်တဲ့ အခါမှာလည်း ကူတယ်

## Example
- YouTube closed captions were built for deaf users but everyone uses them in noisy places
- Sidewalk curb cuts were designed for wheelchairs but help strollers, bikes, and luggage too
- Dark mode was created to reduce eye strain but now everyone uses it at night
- Siri/Alexa started as accessibility tools but became mainstream features

## Example_MM
- Myanmar app တွေမှာ font size ကြီးကြီး ထားတာ — အသက်ကြီးတဲ့ user တွေအတွက် ကူပေမယ့် bus ပေါ်မှာ ဖတ်ရတဲ့ လူငယ်တွေလည်း အဆင်ပြေတယ်
- KBZ Pay မှာ ငွေလွှဲပြီး vibration feedback ပေးတာ — မျက်စိမမြင်သူအတွက်ပေမယ့် phone မကြည့်ဘဲ confirm ရတဲ့ user အားလုံး အဆင်ပြေတယ်
- Myanmar government website တွေ high contrast မသုံးလို့ နေရောင်ကြီးထဲ phone ကြည့်ရင် ဖတ်ရခက်တယ်
- Grab Myanmar app voice command feature က ကားမောင်းနေတဲ့ driver အတွက် accessibility ဖြစ်သလို safety feature လည်း ဖြစ်တယ်
