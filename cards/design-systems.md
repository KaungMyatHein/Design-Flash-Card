# Design Systems
# Design System (ဒီဇိုင်းစနစ်)

## Question
What is a design system and why do teams need one?

## Question_MM
Design system ဆိုတာ ဘာလဲ၊ ဘာကြောင့် team တွေ လိုအပ်တာလဲ?

## Answer
A design system is a living collection of all your product's reusable components — buttons, forms, navigation, grids, and everything made of pixels. It includes both the visual design AND the working code, so when you change the system, you change the product.

## Answer_MM
Design system ဆိုတာ product ရဲ့ reusable component အားလုံးကို စုစည်းထားတာ — button, form, navigation, grid, pixel နဲ့ ဖန်တီးထားတဲ့ အရာအားလုံး ပါတယ်။ Visual design ရော working code ရော ပါတယ်။ System ပြောင်းရင် product ပါပြောင်းတယ်။

## Why
- Eliminates duplicative work across teams — no reinventing the wheel
- Developers can start building without waiting for designers to specify every element
- Designers can focus on new challenges instead of recreating existing components
- Approval cycles speed up because repetitive elements are no longer debated
- Creates consistency across different products and business lines

## Why_MM
- Team တွေကြား duplicative work ကို ဖယ်ရှားတယ် — wheel ပြန်တီထွင်စရာ မလို
- Developer တွေ designer specify လုပ်တာ မစောင့်ဘဲ build စနိုင်တယ်
- Designer တွေ existing component ပြန်ဆောက်မနေဘဲ challenge အသစ်တွေ focus နိုင်တယ်
- Approval cycle မြန်တယ်၊ repetitive element တွေ debate လုပ်စရာ မလိုတော့လို့
- Product အမျိုးမျိုးမှာ consistency ဖန်တီးတယ်

## Example
- Google Material Design provides unified components for all Android apps worldwide
- Shopify Polaris gives merchants a consistent design language across all tools
- Atlassian's design system ensures Jira, Confluence, and Trello feel like one family
- IBM Carbon Design System includes both Figma components and production React code

## Example_MM
- KBZ Bank app တွေ (KBZ Pay, KBZ iBanking) button style တူအောင် design system သုံးရင် user တွေ brand ကို ယုံကြည်တယ်
- Myanmar startup တစ်ခုမှာ designer ၃ ယောက်ရှိရင် design system မရှိဘဲ button ၃ မျိုး ဖြစ်နေတယ်
- Figma component library ဆောက်ထားရင် Myanmar freelance designer တွေ project အသစ်တိုင်း component ပြန်ဆောက်စရာ မလိုဘူး
- Wave Money app ရဲ့ color, typography, spacing ကို system တစ်ခုထဲ define ထားရင် team အသစ်ဝင်လာလည်း consistent design ထုတ်နိုင်တယ်
