# Natural Mapping
# Natural Mapping (သဘာဝ ပုံစံချိတ်ဆက်မှု)

## Question
What is Natural Mapping in design?

## Question_MM
Design မှာ Natural Mapping ဆိုတာ ဘာလဲ?

## Answer
Natural Mapping is when the layout of controls matches the layout of the things they control. It uses spatial relationships so users immediately understand which control affects which element — no labels or instructions needed.

## Answer_MM
Control တွေရဲ့ နေရာချထားပုံက ထိန်းချုပ်တဲ့ အရာတွေရဲ့ နေရာနဲ့ ကိုက်ညီတာ။ ဘယ် control က ဘာကို ထိန်းတယ်ဆိုတာ label မလိုဘဲ ချက်ချင်း နားလည်တယ်။

## Why
- Don Norman: 3 levels of mapping effectiveness
- Best: controls mounted directly on the item
- Second: controls placed close to the item
- Third: controls arranged in the same spatial pattern
- Poor mapping (e.g., stove burners in grid, controls in row) → constant user errors

## Why_MM
- Don Norman က mapping ထိရောက်မှု ၃ ဆင့် ပြောတယ်
- အကောင်းဆုံး: control ကို item ပေါ်မှာ တိုက်ရိုက် ကပ်ထား
- ဒုတိယ: item နားနီးနီးမှာ ချထား
- တတိယ: item တွေရဲ့ pattern အတိုင်း control တွေကို စီထား
- Mapping ညံ့ရင် user အမြဲ မှားမှားနှိပ်နေရတယ် — ဥပမာ မီးဖိုခွက်တွေ grid မှာ ဒါပေမယ့် ခလုတ်တွေ row မှာ

## Example
- Car seat adjustment control shaped like the seat itself
- Light switches arranged in the same pattern as the room's lights
- Volume slider going up for louder, down for quieter
- Car window controls positioned matching the four windows' layout

## Example_MM
- ကားထိုင်ခုံ ချိန်ညှိခလုတ်က ထိုင်ခုံပုံစံ ဖြစ်နေတယ်
- မီးခလုတ်တွေက အခန်းထဲ မီးနေရာအတိုင်း စီထားတယ်
- Volume slider ကို အသံကျယ်ချင်ရင် အပေါ်တွန်း၊ တိတ်ချင်ရင် အောက်ဆွဲ
- ကားပြတင်းပေါက်ခလုတ် ၄ ခုက ပြတင်းပေါက် ၄ ခုရဲ့ နေရာအတိုင်း ချထားတယ်
