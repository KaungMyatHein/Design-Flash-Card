# Gestalt Principle: Similarity
# Gestalt အခြေခံမူ: Similarity (တူညီမှု)

## Question
What is the Gestalt Principle of Similarity?

## Question_MM
Gestalt ရဲ့ Similarity အခြေခံမူဆိုတာ ဘာလဲ?

## Answer
Elements that share visual characteristics — like color, shape, size, or style — are perceived as belonging to the same group, even when they're physically separated.

## Answer_MM
အရောင်၊ ပုံသဏ္ဍာန်၊ အရွယ်အစား သို့မဟုတ် style တူတဲ့ element တွေကို ကွဲကွဲနေပေမယ့် "တစုတည်း" လို့ မြင်ကြတယ်။

## Why
- The brain automatically assumes visual similarity = related
- Can show grouping without needing labels
- Even more effective when combined with Proximity
- Deliberate dissimilarity can grab attention too (breaking the pattern)

## Why_MM
- ဦးနှောက်က ပုံစံတူရင် "ဆက်စပ်တယ်" လို့ အလိုအလျောက် ယူဆတယ်
- Label မလိုဘဲ group ခွဲပြလို့ ရတယ်
- Proximity (အနီးကပ်မှု) နဲ့ ပေါင်းရင် ပိုထိရောက်
- တမင် pattern ချိုးပြီး ကွဲအောင်လုပ်ရင်လည်း attention ဆွဲလို့ ရတယ်

## Example
- All clickable links styled in the same blue color
- Navigation items sharing the same font size and weight
- A red "Delete" button standing out from gray action buttons (dissimilarity)
- Dashboard cards with the same border radius and shadow style

## Example_MM
- Click လို့ရတဲ့ link အကုန်လုံး အပြာရောင် style တူတယ်
- Navigation item တွေ font size နဲ့ weight တူတူ သုံးထားတယ်
- Gray button တွေထဲမှာ "Delete" ခလုတ်ကို အနီရောင်နဲ့ ကွဲထွက်စေတယ် (dissimilarity)
- Dashboard card တွေ border radius နဲ့ shadow style တူတူ ဖြစ်နေတယ်
