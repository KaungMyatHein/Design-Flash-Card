# English: Information Scent
# Myanmar: Information Scent (သတင်းအချက်အလက် အနံ့ခံခြင်း)

## Question
English: What is "information scent" and why does it matter for navigation design?

## Question_MM
User တွေက website တစ်ခုမှာ သူတို့ရှာနေတာကို ဘယ်လိုရှာတွေ့နိုင်လဲ — "Information Scent" ဆိုတာ ဘာလဲ?

## Answer
English: Information scent is the strength of visual and textual cues that tell users they're heading in the right direction. It comes from information foraging theory — just like animals follow a scent trail to find food, users follow link labels, headings, and visual cues to find what they need. Strong scent means users click confidently; weak scent means they hesitate or leave.

## Answer_MM
Information Scent ဆိုတာ user တွေက link တစ်ခု၊ heading တစ်ခုကို ကြည့်လိုက်တာနဲ့ "ဟုတ်ပြီ ဒီဘက်သွားရင် ရှာနေတာ တွေ့မယ်" ဆိုတဲ့ ယုံကြည်မှုပေးတဲ့ အချက်အလက် အနံ့လိုမျိုးပဲ။ တိရစ္ဆာန်တွေက အစားအစာရဲ့ အနံ့ကို ခံပြီး ရှာသလိုပဲ user တွေကလည်း link text တွေ၊ heading တွေကို ကြည့်ပြီး navigate လုပ်တယ်။ အနံ့ကောင်းရင် user ယုံကြည်ချက်ရှိရှိ click လုပ်တယ်၊ အနံ့မရှိရင် ဝေခွဲမရဘဲ site ကနေ ထွက်သွားတယ်။

## Why
- English: Users don't read everything — they scan for clues that match what they're looking for
- English: Clear link labels act like strong scent trails that keep users moving forward with confidence
- English: Vague or clever link text (like "Click here" or "Learn more") gives off weak scent and causes hesitation
- English: Three easy, confident clicks beat one click that makes users stop and think
- English: Good information scent reduces bounce rates and helps users complete tasks faster

## Why_MM
- User တွေက page တစ်ခုလုံး မဖတ်ဘူး — သူတို့လိုချင်တာနဲ့ match ဖြစ်မယ့် clue တွေကိုပဲ scan လုပ်တယ်
- Link text ကရှင်းရင် user တွေက ယုံကြည်ချက်ရှိရှိ ရှေ့ဆက်သွားတယ်
- "Click here" တို့ "Learn more" တို့ လိုမျိုး vague link တွေက user ကို ဝေခွဲမရအောင် လုပ်တယ်
- မစဉ်းစားရဘဲ click ၃ ချက်နှိပ်ရတာက စဉ်းစားရတဲ့ click ၁ ချက်ထက် ပိုကောင်းတယ်
- Information scent ကောင်းရင် bounce rate ကျပြီး user task completion မြန်တယ်

## Example
- English: Amazon product categories use specific labels like "Electronics > Headphones > Over-Ear" — each level gives strong scent about what's inside
- English: Google search results show page titles and URL previews so users can sniff out the most relevant link before clicking
- English: A navigation menu that says "Pricing" gives stronger scent than one that says "Solutions" when users want to know the cost
- English: Breadcrumbs like "Home > Blog > UX Design > Accessibility" let users see the scent trail they followed

## Example_MM
- Amazon မှာ "Electronics > Headphones > Over-Ear" လိုမျိုး category label တွေက ထဲမှာ ဘာရှိလဲ အတိအကျ ပြောပြတယ်
- Google search result မှာ page title နဲ့ URL preview ပြတယ် — click မလုပ်ခင် ဘယ် link က အသုံးဝင်မလဲ user ခန့်မှန်းလို့ရတယ်
- Navigation menu မှာ "Pricing" လို့ ရေးထားရင် "Solutions" ထက် user ရှာနေတဲ့ ဈေးနှုန်းကို ရှင်းရှင်းလင်းလင်း ညွှန်ပြပေးတယ်
- Breadcrumb က "Home > Blog > UX Design > Accessibility" လို user လာခဲ့တဲ့ လမ်းကြောင်းကို ပြန်ပြပေးတယ်
