# English: Default Effect
# Myanmar: Default Effect

## Question
English: Why do most people stick with pre-selected options instead of changing them?

## Question_MM
လူအများစုက ကြိုတင်ရွေးထားတဲ့ option ကို ဘာကြောင့် ပြောင်းမလဲဘဲ ဒီအတိုင်းပဲ သုံးနေတာလဲ?

## Answer
English: The Default Effect is the tendency for people to go with whatever option is pre-selected or pre-set. Because changing requires effort and decision-making, most people simply accept the default. This makes defaults one of the most powerful nudges in design — the default IS the decision for most users.

## Answer_MM
Default Effect ဆိုတာ လူတွေက ကြိုတင်ရွေးချယ်ထားတဲ့ option ကို ဒီအတိုင်း လက်ခံလိုက်တတ်တယ်ဆိုတဲ့ tendency ပဲ။ ပြောင်းဖို့ effort နဲ့ decision လိုတာကြောင့် လူအများစုက default ကိုပဲ ယူလိုက်တယ်။ ဒါကြောင့် default ဆိုတာ design မှာ အပြင်းထန်ဆုံး nudge တစ်ခုပဲ — user အများစုအတွက် default ကိုယ်တိုင်က decision ပဲ။

## Why
- English: Humans naturally take the path of least resistance — changing takes effort
- English: Defaults feel like a recommendation from the system, adding perceived trust
- English: Statistically, users rarely change default settings no matter how many options exist
- English: Smart defaults can guide users toward better outcomes without forcing them
- English: Bad defaults can lead to frustration, like shipping to the wrong address

## Why_MM
- လူတွေက path of least resistance ကိုပဲ သဘာဝအတိုင်း ယူတတ်တယ် — ပြောင်းဖို့ effort လိုတယ်
- Default တွေက system ကနေ recommend လုပ်ထားသလို ခံစားရပြီး trust ပိုရတယ်
- စာရင်းအင်းအရ option ဘယ်လောက်ပဲရှိရှိ user တွေက default setting ကို အရမ်းရှားရှားပဲ ပြောင်းတယ်
- Smart default တွေက user တွေကို မဖိဘဲ ပိုကောင်းတဲ့ outcome ဆီ guide လုပ်ပေးနိုင်တယ်
- Bad default တွေကတော့ frustration ဖြစ်စေနိုင်တယ်၊ address မှားပို့မိတာမျိုး

## Example
- English: Gmail's nudge "Did you mean to attach a file?" is a smart default behavior that saves users from errors
- English: Organ donation rates are dramatically higher in countries where donation is the default opt-in
- English: Living Social sets the default email audience to "everyone" — most users never change it
- English: Cookie consent banners with "Accept All" pre-selected get far more acceptances than opt-in designs

## Example_MM
- Gmail ရဲ့ "File attach လုပ်ဖို့ မေ့သွားလား?" nudge က smart default behavior ပဲ — user တွေကို error ကနေ ကယ်တယ်
- Organ donation rate က donation ကို default opt-in ထားတဲ့ နိုင်ငံတွေမှာ dramatically ပိုမြင့်တယ်
- Living Social က email audience default ကို "everyone" ထားတယ် — user အများစု ပြောင်းမလဲဘူး
- Cookie consent banner မှာ "Accept All" ကို pre-select ထားရင် opt-in design ထက် acceptance အများကြီး ပိုရတယ်
