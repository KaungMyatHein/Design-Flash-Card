# Serial Position Effect
# Serial Position Effect (စာရင်းထဲ နေရာအလိုက် မှတ်မိမှု)

## Question
Why do people remember the first and last items in a list but forget the middle?

## Question_MM
ဘာကြောင့် လူတွေက list ထဲက ပထမဆုံးနဲ့ နောက်ဆုံး item ကို မှတ်မိပြီး အလယ်ကတွေ မေ့သွားကြတာလဲ?

## Answer
The Serial Position Effect means we remember the first items (primacy effect) and last items (recency effect) in a sequence best. The middle gets forgotten. Smart designers put the most important content at the beginning and end of any list, menu, or page.

## Answer_MM
Serial Position Effect အရ sequence ထဲက ပထမဆုံး item (primacy effect) နဲ့ နောက်ဆုံး item (recency effect) ကို အကောင်းဆုံး မှတ်မိတယ်။ အလယ်က မေ့သွားတယ်။ Designer ကောင်းကောင်းက အရေးကြီးဆုံး content ကို list, menu, page ရဲ့ အစနဲ့ အဆုံးမှာ ထားတယ်။

## Why
- First items benefit from extra rehearsal time in working memory
- Last items are still fresh in short-term memory when recalled
- Middle items get squeezed out by both effects
- Navigation menus should place key items at start and end
- Onboarding flows should front-load and close with the most important info

## Why_MM
- ပထမဆုံး item က working memory ထဲ rehearsal time ပိုရတယ်
- နောက်ဆုံး item က recall လုပ်ချိန် short-term memory မှာ fresh ဖြစ်နေသေးတယ်
- အလယ် item တွေ effect နှစ်ခုကြားမှာ ညှပ်ပြီး မေ့ကျတယ်
- Navigation menu မှာ key item ကို အစနဲ့ အဆုံးမှာ ထားသင့်တယ်
- Onboarding flow မှာ အရေးကြီးဆုံး info ကို ရှေ့ဆုံးနဲ့ နောက်ဆုံးမှာ ထားသင့်တယ်

## Example
- iOS bottom tab bar မှာ Home ကို ဘယ်ဘက်ဆုံး၊ Profile ကို ညာဘက်ဆုံး ထားတယ်
- KBZ Pay app မှာ အသုံးအများဆုံး feature ကို tab bar ရဲ့ ပထမနဲ့ နောက်ဆုံးမှာ ထားတယ်
- Presentation မှာ key message ကို intro နဲ့ conclusion မှာ ထားတယ် — middle slide မေ့ကျတယ်
- E-commerce category list မှာ popular category ကို ထိပ်နဲ့ အောက်ဆုံးမှာ ထားတယ်

## Example_MM

- Presentation key message ကို intro နဲ့ conclusion မှာ ထားတယ် — အလယ်က slide မေ့ကျတယ်
- Shopee Myanmar category list မှာ popular category ကို ထိပ်ဆုံးနဲ့ အောက်ဆုံးမှာ ထားတယ်
