# Visual Hierarchy
# Visual Hierarchy (အမြင်အဆင့်ဆင့်စီစဉ်မှု)

## Question
What is Visual Hierarchy and what are its building blocks?

## Question_MM
Visual Hierarchy ဆိုတာ ဘာလဲ၊ ဘာတွေနဲ့ တည်ဆောက်ထားတာလဲ?

## Answer
Visual Hierarchy is the arrangement of design elements to guide the user's eye in order of importance. It uses size, color, contrast, layout, spacing, and style to make some elements stand out more than others so users instantly know what matters most.

## Answer_MM
User ရဲ့ မျက်လုံးကို အရေးကြီးတာကနေ စပြီး လမ်းညွှန်ဖို့ element တွေကို စီစဉ်ထားတာ။ Size, color, contrast, spacing တွေနဲ့ ဘာ အရေးကြီးဆုံးလဲ ချက်ချင်း သိစေတယ်။

## Why
- Important things are more prominent (larger, bolder, distinct color, more whitespace)
- Related things are visually grouped together
- Nested items show parent-child relationships
- Without hierarchy, everything competes equally → users leave
- Building blocks: size, color, layout, spacing, style

## Why_MM
- အရေးကြီးတာကို ပိုထင်ရှားအောင် လုပ်ရတယ် — ပိုကြီး၊ ပိုထူ၊ အရောင်ထူးခြား၊ whitespace များ
- ဆက်စပ်တာတွေကို မျက်စိနဲ့ အုပ်စုတူပုံ ပေါ်အောင် စီထားရတယ်
- အထဲက item တွေက parent-child ဆက်နွယ်မှု ပြရတယ်
- Hierarchy မရှိရင် element အကုန် တန်းတူ ယှဉ်နေတာမို့ user ရှုပ်ပြီး ထွက်သွားတယ်
- တည်ဆောက်တဲ့ အရာတွေ: size, color, layout, spacing, style

## Example
- Headlines bigger than body text to show importance
- Colored call-to-action buttons standing out against neutral backgrounds
- Related content grouped in cards with consistent spacing
- Bold key terms in a paragraph to guide scanning

## Example_MM
- Headline ကို body text ထက် ပိုကြီးအောင် ထားတယ်
- CTA button ကို background နဲ့ ခြားပြီး အရောင်ထည့်ထားတယ်
- ဆက်စပ်တဲ့ content တွေကို card ထဲမှာ spacing ညီညီနဲ့ စုထားတယ်
- Paragraph ထဲက အဓိကစာလုံးတွေကို bold လုပ်ပြီး scan လုပ်ရလွယ်အောင် ဆွဲထားတယ်
