# Gestalt Figure-Ground
# Gestalt Figure-Ground (ပုံနဲ့နောက်ခံ နိယာမ)

## Question
How does our brain decide what's the main object and what's the background in a design?

## Question_MM
ငါတို့ ဦးနှောက်က design ထဲမှာ ဘာက main object ဖြစ်ပြီး ဘာက background ဖြစ်တယ်ဆိုတာ ဘယ်လို ဆုံးဖြတ်တာလဲ?

## Answer
Figure-Ground says our brain instinctively separates what we see into a main object (figure) and a background (ground). Designers use this to make important elements pop forward while pushing less important stuff into the background.

## Answer_MM
Figure-Ground principle အရ ဦးနှောက်က မြင်ရတာကို main object (figure) နဲ့ background (ground) အဖြစ် အလိုအလျောက် ခွဲတယ်။ Designer တွေက ဒါကို အသုံးချပြီး အရေးကြီးတဲ့ element ကို ရှေ့ထွက်လာအောင်၊ မအရေးကြီးတာကို နောက်ခံမှာ ရှိနေအောင် လုပ်တယ်။

## Why
- The brain automatically creates depth from flat screens
- Overlapping elements, shadows, and blur create figure-ground separation
- Modal dialogs use this — content becomes figure, dimmed page becomes ground
- Contrast between figure and ground determines visual clarity
- Ambiguous figure-ground can confuse users or create clever optical illusions

## Why_MM
- ဦးနှောက်က flat screen ကနေ depth ကို အလိုအလျောက် ဖန်တီးတယ်
- Overlap, shadow, blur တွေက figure-ground ခွဲခြားမှုကို ဖန်တီးတယ်
- Modal dialog မှာ content က figure ဖြစ်ပြီး dim ဖြစ်တဲ့ page က ground ဖြစ်တယ်
- Figure နဲ့ ground ကြား contrast က visual clarity ကို ဆုံးဖြတ်တယ်
- Figure-ground ရှင်းရှင်းမခွဲနိုင်ရင် user တွေ ရှုပ်ထွေးနိုင်တယ်

## Example
- YouTube's video player popup dims the entire background page
- Material Design cards use shadow elevation to float above the surface
- Apple's hero sections overlay white text on dramatic product photography
- Google Maps dropdown search results appear above the map content

## Example_MM
- KBZ Pay ငွေလွှဲ confirm popup ပေါ်ရင် နောက်ခံ screen dim ဖြစ်တယ် — user focus ကို popup ပေါ်ဆွဲတယ်
- Shopee Myanmar product card တွေမှာ shadow ထည့်ထားလို့ background ကနေ ခွဲထွက်ပြီး float ဖြစ်သလို ခံစားရတယ်
- Myanmar e-commerce site hero banner မှာ product photo အပေါ် "30% OFF" text overlay တင်ထားတယ်
- Grab Myanmar app မှာ destination dropdown ပေါ်ရင် map အပေါ်မှာ ထပ်ပြီး ပေါ်လာတယ်
