# F-Pattern & Z-Pattern
# F-Pattern နဲ့ Z-Pattern (မျက်လုံးလှုပ်ရှားမှုပုံစံများ)

## Question
What are the F-Pattern and Z-Pattern in web design?

## Question_MM
Web design မှာ F-Pattern နဲ့ Z-Pattern ဆိုတာ ဘာတွေလဲ?

## Answer
F-Pattern and Z-Pattern are the natural eye movement paths users follow when scanning web pages. F-Pattern happens on text-heavy pages; Z-Pattern happens on open, visual pages.

## Answer_MM
User တွေ web page ကြည့်ရင် မျက်လုံးက သဘာဝအတိုင်း သွားတဲ့ လမ်းကြောင်းတွေ။ F-Pattern က စာများတဲ့ page မှာ ဖြစ်တယ်၊ Z-Pattern က ပုံတွေ space တွေ များတဲ့ page မှာ ဖြစ်တယ်။

## Why
- Eye tracking studies show user scanning patterns are predictable
- F-Pattern: scan across → go down left side → scan again if something catches the eye
- Z-Pattern: scan right → diagonal to bottom-left → scan right again → zigzag continues
- Knowing these helps you place key content exactly where users will look

## Why_MM
- Eye tracking study တွေက user ကြည့်ပုံကို ကြိုတင်ခန့်မှန်းလို့ရကြောင်း ပြတယ်
- F-Pattern: အပေါ်တန်း ဖတ် → ဘယ်ဘက်ကပ်ပြီး ဆင်း → စိတ်ဝင်စားတာတွေ့ရင် ထပ် ဖတ်
- Z-Pattern: ညာဘက် scan → ဘယ်အောက်ထောင့် ကူး → ညာဘက် ထပ် scan → zigzag ဆက်
- ဒါတွေ သိရင် CTA, heading, key info ကို user မြင်မယ့် နေရာမှာ တိတိကျကျ ချထားလို့ရတယ်

## Example
- Blog posts placing key info in the first two lines of each paragraph (F-pattern)
- Landing pages placing logo top-left, CTA top-right, content bottom-left (Z-pattern)
- News sites with headlines aligned left for easy F-pattern scanning
- Marketing pages with zigzag image-text alternating layout

## Example_MM
- Blog post မှာ paragraph ရဲ့ ပထမ ၂ ကြောင်းမှာ key info ထည့်ထားတယ် (F-pattern)
- Landing page မှာ logo ဘယ်အပေါ်၊ CTA ညာအပေါ်မှာ ထားတယ် (Z-pattern)
- News site မှာ headline တွေကို ဘယ်ဘက် ကပ်ထားတယ်
- Marketing page မှာ ပုံနဲ့ စာကို zigzag ပုံစံ တလှည့်စီ ချထားတယ်
