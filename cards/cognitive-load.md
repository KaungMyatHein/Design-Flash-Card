# Cognitive Load
# Cognitive Load (စိတ်ပိုင်းဆိုင်ရာ ဝန်)

## Question
What is cognitive load and why should designers care about reducing it?

## Question_MM
Cognitive load ဆိုတာ ဘာလဲ၊ ဘာကြောင့် designer တွေက လျှော့ချဖို့ ဂရုစိုက်သင့်တာလဲ?

## Answer
Cognitive load is the amount of mental effort your brain needs to process information. When an interface demands too much thinking, remembering, or deciding, users get overwhelmed and make mistakes. Good design minimizes this strain.

## Answer_MM
Cognitive load ဆိုတာ ဦးနှောက်က information process လုပ်ဖို့ လိုတဲ့ mental effort ပမာဏ ဖြစ်တယ်။ Interface က အတွေးများတာ၊ အမှတ်ရရတာ၊ ဆုံးဖြတ်ရတာ များလွန်းရင် user တွေ overwhelm ဖြစ်ပြီး အမှားလုပ်မိတယ်။ Design ကောင်းကောင်းက ဒီ strain ကို minimize လုပ်တယ်။

## Why
- Cognitive load is the most "expensive" type of mental processing
- Users already have a lot on their mind — don't add more
- Extra clicks are better than extra thinking (motor < cognitive)
- Mixing text, images, and graphics creates rhythm that eases cognitive strain
- Stressed users have reduced working memory capacity

## Why_MM
- Cognitive load က mental processing အမျိုးအစားတွေထဲ အ "စျေးကြီး" ဆုံး ဖြစ်တယ်
- User တွေ ခေါင်းထဲမှာ တွေးစရာ အများကြီးရှိပြီးသား — ထပ်မထည့်ပါနဲ့
- Click ပိုများတာက တွေးရတာ ပိုများတာထက် ပိုကောင်းတယ် (motor < cognitive)
- Text, image, graphic ရောထားရင် rhythm ဖြစ်ပြီး cognitive strain လျော့တယ်
- Stress ရှိတဲ့ user တွေရဲ့ working memory capacity ကျတယ်

## Example
- Google changes visited link colors to purple so users don't revisit the same pages
- Amazon's product comparison table lets you view specs side by side without switching tabs
- Chrome autofill suggests addresses and emails so you don't have to type them
- Stripe dashboard uses charts instead of raw numbers to show revenue trends at a glance

## Example_MM
- KBZ Pay မှာ ငွေလွှဲဖို့ step ၃ ခုပဲ ရှိတယ် — phone number, amount, confirm — ခေါင်းတွေးစရာ နည်းတယ်
- Shopee Myanmar မှာ recent search ပြထားတယ် — ဘာရှာခဲ့လဲ မှတ်စရာမလိုဘူး
- Wave Money top-up မှာ amount preset button (5000, 10000, 20000) ထားတယ် — ဂဏန်း type လုပ်စရာ မလို
- Myanmar online shop Facebook page တွေမှာ product photo မှာ ဈေးပါ ရေးထားတယ် — DM မေးစရာ မလိုအောင်
