# Doherty Threshold
# Doherty Threshold (တုံ့ပြန်မှု အမြန်နှုန်း)

## Question
How fast should a system respond to keep users feeling in control?

## Question_MM
User တွေ control ရှိနေတယ်လို့ ခံစားရအောင် system က ဘယ်လောက်မြန်မြန် respond လုပ်သင့်လဲ?

## Answer
The Doherty Threshold says system response should be under 400 milliseconds to keep users engaged. At 0.1 seconds, users feel direct control. At 1 second, they notice a delay but stay focused. Beyond 10 seconds, they lose attention completely.

## Answer_MM
System response က 400 milliseconds အောက်ဖြစ်ရင် user တွေ engage ဖြစ်နေမယ်။ 0.1 စက္ကန့်ဆို direct control ခံစားရတယ်။ 1 စက္ကန့်ဆို delay သတိထားမိပေမယ့် focus မပျက်ဘူး။ 10 စက္ကန့်ကျော်ရင်တော့ attention လုံးဝပျက်သွားတယ်။

## Why
- Fast response keeps users in a productive flow state
- Even a few hundred milliseconds of delay breaks momentum
- Users start losing their feeling of control after 1-2 seconds
- Progress bars and loading animations help bridge unavoidable delays
- Speed is a feature — users equate fast response with quality

## Why_MM
- Response မြန်ရင် user တွေ productive flow state မှာ ဆက်နေတယ်
- Milliseconds အနည်းငယ် delay ရှိရုံနဲ့ momentum ပျက်နိုင်တယ်
- 1-2 စက္ကန့်ကျော်ရင် user တွေ control ပျက်တဲ့ ခံစားချက်ရတယ်
- Progress bar နဲ့ loading animation က မလွှဲမရှောင်သာ delay ကို ခံနိုင်အောင် ကူတယ်
- Speed က feature တစ်ခု — user တွေက မြန်တာကို quality နဲ့ ချိတ်ဆက်ပြီး ခံစားတယ်

## Example
- Google returns search results in under 0.2 seconds — feels instant
- Instagram "like" heart animation gives immediate visual feedback (0.1s)
- Dropbox shows upload progress bars so users know the system is working
- Facebook uses skeleton screens to make content loading feel seamless

## Example_MM
- KBZ Pay ငွေလွှဲတဲ့အခါ loading spinner ပြတယ် — ဒါပေမယ့် ၃ စက္ကန့်ကျော်ရင် user တွေ "ငွေရောက်ပြီလား" ဆိုပြီး စိတ်ပူတယ်
- Grab Myanmar မှာ driver ရှာနေတဲ့ animation ပြတယ် — wait time ကို visually ခံနိုင်အောင် လုပ်ပေးတယ်
- Food Panda order status ကို real-time update ပြတယ် — "ချက်နေတယ်" "ပို့နေတယ်" ပြလို့ user စိတ်ချရတယ်
- Shopee Myanmar page load နှေးရင် user တွေ app ပိတ်ပြီး Lazada ကို ပြောင်းသွားတယ်
