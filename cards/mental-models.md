# Mental Models
# Mental Models (စိတ်ပိုင်းဆိုင်ရာ ပုံစံများ)

## Question
What are Mental Models and why do they matter in UX?

## Question_MM
Mental Models ဆိုတာ ဘာတွေလဲ၊ UX မှာ ဘာကြောင့် အရေးကြီးတာလဲ?

## Answer
Mental models are the beliefs and expectations users already have about how things work. When your design matches users' mental models, it feels intuitive. When it doesn't, users get confused.

## Answer_MM
User တွေမှာ အရာတွေ ဘယ်လိုအလုပ်လုပ်တယ်ဆိုတဲ့ ယုံကြည်ချက်နဲ့ မျှော်လင့်ချက် ရှိပြီးသားရှိတယ်။ Design က အဲ့ဒါနဲ့ ကိုက်ရင် intuitive ခံစားရတယ်။ မကိုက်ရင် ရှုပ်သွားတယ်။

## Why
- Users build schemas (knowledge frameworks) from past experience
- Don Norman: conceptual model (designer's intent) must match mental model (user's understanding)
- Info that fits existing schemas is much easier to learn
- Discoverable design helps build correct mental models
- Without user research, guessing mental models often goes wrong

## Why_MM
- User တွေ အရင် experience ကနေ schema (ဗဟုသုတ ဘောင်) ဆောက်ထားကြတယ်
- Don Norman: conceptual model (designer ရည်ရွယ်တာ) နဲ့ mental model (user နားလည်တာ) ကိုက်ညီရမယ်
- ရှိပြီးသား schema ထဲ ကိုက်တဲ့ info က သင်ယူရတာ အများကြီး လွယ်တယ်
- ရှာဖွေလို့ ရတဲ့ (discoverable) design က mental model မှန်မှန် ဆောက်ပေးတယ်
- User research မလုပ်ဘဲ mental model ကို ခန့်မှန်းရင် တော်တော်များများ မှားတယ်

## Example
- Shopping cart icon on e-commerce (everyone knows what it means)
- Hamburger menu icon for navigation on mobile
- Swipe to delete on iOS (matches physical "push away" mental model)
- Folder/file metaphor in operating systems

## Example_MM
- E-commerce မှာ shopping cart icon — လူတိုင်း ဘာလဲ သိတယ်
- Mobile navigation မှာ hamburger menu icon
- iOS မှာ swipe to delete — ပစ္စည်းကို လက်နဲ့ "တွန်းဖယ်" တာနဲ့ တူတယ်
- Computer မှာ folder/file ပုံစံ — တကယ့် ဖိုင်တွဲ/စာရွက်နဲ့ တူအောင် လုပ်ထားတာ
