# English: Von Restorff Effect (Isolation Effect)
# Myanmar: Von Restorff Effect (ထူးခြားတဲ့ဟာ မှတ်မိတယ်)

## Question
English: Why do users remember and notice one specific element more than others on a screen?

## Question_MM
Screen ပေါ်မှာ element တွေအများကြီးထဲက တစ်ခုကိုပဲ user က ပိုမှတ်မိတာ ဘာကြောင့်လဲ?

## Answer
English: The Von Restorff Effect says that when multiple similar items are presented together, the one that stands out as different will be remembered the most. In UX design, you can use this by making your most important element — like a CTA button or a key message — visually distinct from everything else on the page.

## Answer_MM
Von Restorff Effect ဆိုတာ ဆင်တူတဲ့ item တွေအများကြီးထဲမှာ ထူးခြားနေတဲ့ တစ်ခုကိုပဲ လူတွေက ပိုမှတ်မိတယ်ဆိုတဲ့ psychology principle ပါ။ UX design မှာ ဒီ principle ကို သုံးပြီး CTA button တို့ key message တို့ကို တခြား element တွေနဲ့ မတူအောင် ခြားပြီးပြလို့ရတယ်။

## Why
- English: Our brains are wired to notice things that break a pattern — the odd one out grabs attention automatically
- English: Consistency creates a baseline so that one deliberate inconsistency becomes a powerful spotlight
- English: Too many standout elements cancel each other out — save distinctiveness for what truly matters
- English: This effect works across color, size, shape, and position — any visual difference can trigger it
- English: It directly helps users find the most important action without scanning the whole page

## Why_MM
- ဦးနှောက်က pattern ကျိုးတဲ့ဟာကို အလိုအလျောက် သတိထားမိတယ်
- Consistency ရှိနေမှ inconsistency တစ်ခုက ထိထိရောက်ရောက် spotlight ဖြစ်တယ်
- Standout element များရင် ဘယ်ဟာမှ မထင်ရှားတော့ဘူး — အရေးကြီးတဲ့ဟာအတွက်ပဲ သုံးပါ
- Color, size, shape, position — ဘယ် visual difference မဆို ဒီ effect ကို trigger လုပ်နိုင်တယ်
- User က page တစ်ခုလုံး scan မနေဘဲ အရေးကြီးတဲ့ action ကို တန်းတွေ့အောင် ကူညီတယ်

## Example
- English: A bright orange "Sign Up" button surrounded by gray navigation links on a landing page
- English: Spotify's green "Play" button stands out among the muted dark interface elements
- English: Pricing pages highlight the "recommended" plan with a different color or border to draw attention
- English: Error messages in red text immediately stand out from black body text on a form

## Example_MM
- Landing page မှာ navigation link တွေက gray ဖြစ်ပြီး "Sign Up" button ကိုပဲ အရောင်တောက်တောက်ထားတယ်
- Spotify ရဲ့ dark interface ထဲမှာ green "Play" button က တစ်ခုတည်းထင်ရှားနေတယ်
- Pricing page တွေမှာ "Recommended" plan ကို တခြား plan တွေနဲ့ အရောင်ခြား border ခြားပြီး ခွဲပြတယ်
- Form ထဲမှာ error message ကို red နဲ့ပြတာနဲ့ black text တွေထဲက ချက်ချင်းမြင်ရတယ်
